FAQ

Is the data discoverable?

The data is not discoverable,it is immediately deleted after the data processing is completed.

What is happening with my data?

While you're uploading the full conversation of your chat, I am saving only numbers and data that will require to show you the analytics.

Which type of data are you saving?

Mostly I'm saving only numbers and summaries of each section. For example, I'm saving the total number of messages sent, but I'm not saving the messages's content itself. I do save things like common emojis, words, and active users in order to show you some nice visualizations over this data.

What does the exported text file contains?

The text file you're exporting from WhatsApp contain a list of messages from your chat. The format of the file is *.txt and each line presents a message,the date and time at which it was sent and the name of person who sent it in the chat.

When I uploaded my chat, some graphs are showing wrong information ?

This Chat analyzer assumes Time in 24-hour format and Date in English(UK) format.
